# 🔐 Animated Sign-In & Sign-Up Form

A beautiful, modern, and fully responsive **Sign-In and Sign-Up form** with smooth sliding animations. This project is built using semantic HTML, custom CSS properties, and JavaScript to handle the toggle transitions between forms.

## ✨ Features
* **Smooth Transitions:** Sliding animation between Sign In and Create Account states.
* **Modern UI/UX:** Clean typography, balanced shadows, and a subtle color palette.
* **Social Authentication Icons:** Includes buttons for Google, Facebook, GitHub, and LinkedIn login options.
* **Form Validation Ready:** Built with native input types for email and password fields.
* **Responsive Design:** Works flawlessly across different screen sizes.

---

## 🛠️ Tech Stack Used
* **HTML5:** For structured and semantic markup.
* **CSS3:** For styling, flexbox/grid layout, and custom animations.
* **JavaScript:** To toggle active classes and trigger form switching.
